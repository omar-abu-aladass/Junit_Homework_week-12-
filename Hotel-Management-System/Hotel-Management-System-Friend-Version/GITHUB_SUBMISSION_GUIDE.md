# GitHub Upload Guide

Follow these steps after opening Git Bash inside the project folder.

## Suggested repository name

```text
Hotel-Management-System-UML-Refactor
```

## Git commands

```bash
git init
git add .
git commit -m "Add UML class diagram and refactored Java code"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/Hotel-Management-System-UML-Refactor.git
git push -u origin main
```

Replace `YOUR_USERNAME` with the GitHub username.
